# 📖 Overview

**Android Media Permission Demo** is a simple Android application developed using Java that demonstrates how to request and manage Android 13+ granular media permissions.

This project showcases the implementation of:

- 📸 `READ_MEDIA_IMAGES`
- 🎥 `READ_MEDIA_VIDEO`
- 🎵 `READ_MEDIA_AUDIO`

instead of the deprecated `READ_EXTERNAL_STORAGE` permission for secure media file access on Android 13 and higher devices.

Perfect for:

- Android Developers
- Android 13 Permission Learning
- Storage Permission Handling
- Mobile App Development
- Android Security Best Practices

# 📂 Permissions Used

```xml
<!-- Android 13+ Media Permissions -->

<uses-permission android:name="android.permission.READ_MEDIA_VIDEO"/>
```
